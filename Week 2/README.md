## Week 2: Relational Database Design and SQL Aggregations

This week focused on storing the cleaned dataset in a relational database and performing data analysis using SQL queries.

### Key Activities:

* Created a database and table in MySQL
* Imported the cleaned dataset into the database

### SQL Analysis Performed:

* Calculated Total Revenue using SUM()
* Computed Average Order Value using AVG()
* Counted total orders using COUNT()
* Identified best-selling products using GROUP BY and ORDER BY
* Analyzed revenue distribution by city and category
* Evaluated sales performance across different channels

### Tools Used:

* MySQL (for database management and querying)
* GitHub (for storing SQL scripts)

### Outcome:

Meaningful insights were extracted from the dataset using SQL, providing a strong foundation for dashboard development in the next phase.
